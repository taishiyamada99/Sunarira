# Contributor Code of Conduct

## Our Commitment

We want Sunarira to be a welcoming and respectful community for everyone,
regardless of background, experience level, identity, or personal
circumstances.

## Expected Behavior

- Be respectful and constructive in issues, discussions, and pull requests.
- Assume good intent and provide actionable technical feedback.
- Focus on facts and impact, not personal attacks.
- Accept that maintainers may prioritize stability, security, and scope.

## Unacceptable Behavior

- Harassment, hate speech, discrimination, or abusive language.
- Personal attacks, intimidation, or sustained disruptive behavior.
- Sharing private information without consent.
- Deliberately harmful technical guidance or exploit posting without
  responsible disclosure.

## Enforcement

Project maintainers may edit, hide, lock, or remove content that violates this
Code of Conduct. Repeated or severe violations may result in temporary or
permanent exclusion from project spaces.

## Reporting

If you experience or witness unacceptable behavior, report it through a private
maintainer channel when possible. For security-related findings, use the
security process described in [SECURITY.md](SECURITY.md).

## Scope

This Code of Conduct applies to repository spaces (issues, pull requests,
discussions, reviews) and to project-related communication channels.
