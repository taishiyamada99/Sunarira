# Contributing

Thanks for contributing to Sunarira.

## How to Contribute

- Open an issue first for bug reports, UX changes, or larger proposals.
- For code changes, open a pull request with a clear summary and test notes.
- Keep changes focused and include or update tests when behavior changes.

## Development Checks

- Run tests before opening a pull request:
  - `xcodebuild test -project Sunarira.xcodeproj -scheme Sunarira -configuration Debug -derivedDataPath build/DerivedData`
- Confirm local environment requirements in [README.md](README.md) (`Requirements` section).
